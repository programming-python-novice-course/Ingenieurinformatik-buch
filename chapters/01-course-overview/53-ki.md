# Einsatz von LLMs (z. B. ChatGPT) beim Programmieren
Künstliche Intelligenz kann Sie beim Programmieren sinnvoll unterstützen, wenn Sie sie gezielt und kritisch einsetzen.

- Sie können LLMs über eine Website nutzen (z. B. ChatGPT im Browser).
- Sie können LLMs direkt in der Entwicklungsumgebung als KI-Agent nutzen, sodass kein ständiges Kopieren von Code nötig ist und die KI direkten Projektbezug hat.


Sie können KI besonders sinnvoll einsetzen, wenn Sie die Grundlagen aus **Teil 1** und **Teil 2** verstanden haben. Ohne Konzeptverständnis können Sie der KI oft nicht präzise sagen, *was* Sie brauchen und *unter welchen Randbedingungen* die Lösung funktionieren muss.


## Aus Erfahrung

- In etwa **70 %** der Fälle erzeugt KI schnell sinnvolle, kleinere Code-Abschnitte, die funktionieren und oft direkt eingesetzt werden können.
- In den übrigen **30 %** wirken Lösungen zunächst korrekt, sind aber nicht stabil/robust oder passen nicht zum Gesamtsystem.

Der entscheidende Punkt ist: In der Praxis werden Sie für die **30 %** bezahlt, nicht für die **70 %** (das kann jede Person mit LLM-Zugang). Deshalb lernen Sie:

- Randbedingungen mitzudenken
- Fehlerquellen vorherzusehen
-  mit komplexen, gewachsenen Systemen zu arbeiten (z. B. Sicherheitsanforderungen, unvollständige Dokumentation, Altcode)

```{admonition} Merksatz
::class: tip

- Voraussetzung für sinnvolle KI-Nutzung ist Konzeptverständnis aus **Teil 1** und **Teil 2** sowie ein systematisches Vorgehen (insbesondere **Teil 4**).
- Selbst wenn Sie sich später bei der Syntax (Teil 3) KI-unterstützen lassen, müssen Sie Syntax gut genug verstehen, um semantische Fehler in eigenem, KI-generiertem oder fremdem Code erkennen zu können.
```
