# Gliederung der Vorlesungsinhalte


Die Inhalte der Vorlesung sind in mehrere Kapitel unterteilt. 

| Kapitel | Lernziele | Inhalte |
|---------|----------|---------|
| **01: Kursübersicht** | Studierende sollen die Kursstruktur und Arbeitsmaterialien kennen. | Arbeitsmaterialien, Prüfungsordnung, Kursinhalte, Lesehinweise |
| **02: Kurskonzept** | Studierende sollen verschiedene Softwaretypen unterscheiden können und den Stellenwert der Programmierung im Softwareentwicklungsprozess verstehen. | Softwaretypen, Softwareentwicklungsprozess, Programmiersprachen (Syntax, Semantik, Konzepte), Erfolgskriterien, Fallbeispiel, Gliederung |
| **03: Basiswissen Informatik** | Studierende sollen Algorithmen, Programmcode, Pseudocode und Programme definieren können. Sie sollen Programmierkonstrukte und Datenstrukturen kennenlernen und anwenden können. | Grundlagen der Informatik, Algorithmen, Programmieren, Wiederholung als Grundprinzip, Kontrollstrukturen, Datenstrukturen |
| **04: Spracheigenschaften** | Studierende sollen die Eigenschaften der Programmiersprache Python verstehen und deren Auswirkungen auf die Programmierung erkennen können. | Interpretation/Kompilierung, Programmierparadigmen, Syntax, Referenzen, Typisierung, primitive Datentypen |
| **05: Python Einführung** | Studierende sollen die Grundlagen der Python-Programmierung verstehen. | Namenskonventionen (PEP 8), Python-Philosophie |
| **06: Python Ökosystem und Setup** | Studierende sollen das Python-Ökosystem nutzen können, Python installieren und ausführen können sowie Pakete, Module und Jupyter Notebooks verwenden können. | Installation, Ausführung von Python-Programmen, Jupyter Notebooks, Module und Pakete |
| **07: Python Grundlagen** | Studierende sollen den Zusammenhang zwischen Variablen und Arbeitsspeicher verstehen und die Auswertung von Funktionsaufrufen und Ausdrücken nachvollziehen können. | Variablen, Ausdrücke, Funktionsaufrufe |
| **08: Python Datentypen** | Studierende sollen verschiedene Datentypen in Python unterscheiden und anwenden können. | Datentypen-Grundlagen, int, float, bool, list, tuple, string, set, dict, range, Type Hints |
| **09: Funktionen** | Studierende sollen Funktionen definieren und verwenden können sowie Rekursion verstehen und anwenden können. | Reine Funktionen, Funktionsdefinition, Namespaces, Lambdas, verschachtelte Funktionen, Rekursion, First-Class Functions |
| **10: Kontrollstrukturen** | Studierende sollen Fallunterscheidungen und Schleifen anwenden können. | Fallunterscheidungen (if/else), Schleifen (for/while), Comprehensions |
| **11: Objektorientierte Programmierung** | Studierende sollen Objekte und Klassen verstehen, erstellen und anwenden können. | Klassen und Objekte, Vererbung, Polymorphismus, Kapselung |
| **12: Fallbeispiel** | Studierende sollen das erlernte Wissen praktisch anwenden können, sowohl mit Bibliotheken als auch durch Eigenentwicklung. | Lösung mit Bibliotheken, Eigenentwicklung, Software Testing |
