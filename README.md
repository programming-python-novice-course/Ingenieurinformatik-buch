# Open educational resources for Python programming in non-computer-science degree programs - cleaned version for A/S/P analysis

This version is a cleaned version for the A/S/P analysis in the paper Knowledge Markers: An AI-Agnostic Concept for the Design of Programming Courses.

The original version is used as teaching artefact but the in the original:
- some labels were missing
- only units at the finest granularity were labelled which is not useful when high-level-sections provide mental models.

We changed the labeling procedure, allowing also higher-level-sections to be labelled.

As a consequence, we relabbelled the chapters, sections, and units.

This repository contains:
- the source code for the pdf-script without knowledge markers that serves as input for the evaluation.
- a table that gives an overview how instructors and an LLM labelled the contents according to the decision tree introduced in Section 3 of the paper.

The chapters, sections, sub-sections in the table has been derived from the table of contents.
